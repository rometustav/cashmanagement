# Project G11: SME Loan approval prediction

The aim of this project is to predict the loan amount a company got approved for.

2 different prediction models: one tree-based model + one linear regression model

## Training data
SBAnational.csv - dataset consisting of 899164 rows and 27 columns. The dataset is from the U.S. Small Business Administration (SBA).

Prepared_data.csv - dataset after preparation, consisting of 349641 rows and 89 columns.
